# AI Assistance During Development

Artificial Intelligence was used as a development assistant throughout this project.

AI assisted with the following tasks:

- Setting up the React project using Vite.
- Explaining React concepts such as components, props, useState, and useEffect.
- Generating React components including Navbar, NoteInput, and NoteCard.
- Implementing note creation, editing, deletion, searching, and local storage.
- Helping identify and fix import errors and React component issues.
- Suggesting improvements to the project structure and code organization.
- Providing guidance for Git, GitHub, commits, and project deployment.

The AI-generated code was reviewed, tested, and modified where necessary before being included in the final application.
